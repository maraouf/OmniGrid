# TOTP-based 2FA for local-auth users (#345). pyotp implements RFC 6238

# Optional ICMP transport for the Ping host-stats provider (#343).
# logic/ping.py imports this LAZILY and falls back to TCP-connect
# probes when the package is missing — so this dep is genuinely
# optional from a correctness standpoint. We pin it anyway so the
# Settings → Host stats → Ping panel's "use ICMP" toggle is wired
# out of the box; without it the operator sees the
# "icmplib package not installed" hint and the toggle is disabled.
# Runtime caveat: ICMP echo requires CAP_NET_RAW on the container.
# The default python:3.12-slim image does NOT carry the capability;
# enable it by adding ``cap_add: [NET_RAW]`` to the OmniGrid service
# in docker-compose.yml. Without the cap the import succeeds but
# every probe raises SocketPermissionError → ping.py auto-falls-back
# to TCP at probe time, so a missing cap is not a boot-blocker.
icmplib==3.0.4
