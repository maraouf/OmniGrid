# TOTP-based 2FA for local-auth users (#345). pyotp implements RFC 6238

# from SESSION_SECRET so rotating that env var rotates every stored
# secret). cryptography is already pulled in transitively by PyJWT[crypto]

# logic/totp.py has a hard contract on the Fernet API.
